Para añadir un módulo nuevo (Básico, RazasYCulturas, Personajes, ...) descomprimir el archivo ZIP descargado de la página oficial (librodeesher.sourceforge.net) y guardarlo en la carpeta módulos.
Después ejecutar el programa y en opciones incluir el path en el campo indicado. 


Existe un problema conocido con los módulos. Normalmente se debe a que los ficheros con las características y las habilidades se bajan en un formato ilegible para tu ordenador (ocurre en Windows en la mayoría de ordenadores). Si te pasa este problema (lo reconocerás por que al abrir el fichero rolemaster/características.txt los acentos se ven de forma extraña), visita la página web del Libro de Esher y descargate el fichero rolemaster_win.zip y sustituye el contenido del programa por el de este fichero.
